  1. Linux配置IP地址的方法
  • ifconfig：查看与配置网络命令状态
    例：ifconfig eth0 192.168.0.200 netmask 255.255.255.0 #临时配置eth0网卡的ip和子网掩码
  1. Linux网络配置文件
  • /etc/sysconfig/network-scripts/ifcfg-eth0：网卡信息文件
  • /etc/sysconfig/network：主机名文件
  • /etc/susconfig/resolv.conf:DNS配置文件
  1. 常用的网络命令
  • ifconfig：查看与配置网络命令状态
  • hostname：主机名
  • ifdown/ifup：关闭与启动网卡
  • netstat：查看网络状态

    -t：列出TCP协议端口
    -u：列出UDP协议端口
    -n：不是用域名与服务名，而使用IP地址和端
    l：仅列出在监听状态网络服务
    a：列出所有的网络连接

  • route -n：查看路由列表（可以查看网关）
  • nslookup [主机名或IP]:进行域名与IP地址解析
  • ping：查看网络连接是否相同，底层是ICMP协议
  • traceroute：路由查看命令，用来查看数据到达某个主机经过的路由地址
