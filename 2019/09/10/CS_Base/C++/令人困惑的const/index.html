令人困惑的const

发表于 2019-09-10 更新于 2019-09-19

const限定符是C++中令人困惑关键字之一。
简单地来说，当一个变量被const限定，它就成为了常量，其值就无法被改变。

const的简单使用

当你想声明一个无法被更改的变量（常量）时，在其声明前加const，但是要注意声明一个常量的同时必须要完成其初始化。个人的理解是，一个声明但没有初始化的常量，在之后无法改变其值（因为是常量），没有意义，因此常量声明的同时必须初始化。

const与指针

在讲解const与指针前，首先我们需要明确指针的含义，指针本身是一个对象，它又可以指向另外一个对象，因此指针包含两部分：指针本身的地址和指针所存的地址（指针所指的对象）。因此const和指针结合使用产生了两个重要的问题：指针本身是不是常量和指针所指的对象是不是常量。

const函数

当我们在函数声明的结尾处加上const，说明该函数为const函数。如果我们试图通过该函数更改对象的成员变量将会引发错误，但是我们可以通过const函数访问对象的成员变量。
