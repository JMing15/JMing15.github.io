Linux-Shell编程

发表于 2019-10-05 更新于 2019-11-15

Shell是什么？

Shell是一个命令行解释器，它为用户提供了一个向Linux内核发送请求以便运行程序的界面系统级程序，用户可以用Shell来启动、挂起、停止甚至编写一些程序。

基本语法

变量

  • 定义变量：变量=值
  • 撤销变量：unset 变量
  • 声明静态变量：readonly 变量，注意：不能unset

注释

  • #!我被注释了
  • :<<! 我被注释了 !

    变量定义规则

  • 变量名称可以有字母、数字、下划线组成，但是不能以数字开头
  • 等号两侧不能有空格
  • 变量名称一般习惯大写

    将命令的返回值赋给变量

  • A=ls -l，运行反引号里面的命令，并把结果返回值赋给变量A
  • A=$(ls -l)，等价于反引号

设置环境变量

  • export 变量名=值：将shell变量输出为环境变量
  • source 配置文件：让修改过的配置信息立即生效
  • echo $变量名：查询环境变量的值

预定义变量

  • $$ 当前进程的进程号PID
  • $! 后台运行的最后一个进程的进程号PID
  • $? 最后一次执行命令的返回状态。如果这个变量的值为0，证明上一个命令被执行；如果这个变量非0（具体为何值，由命令自身决定），则证明上一个命令执行不正确

运算

  • $((运算式)) $[运算式]
  • expr *,/.%

判断语句

  • [ condition ] condition前后有空格

  • 字符串比较
    =

  • 两个整数的比较
    -lt 小于
    -le 小于等于
    -eq 等于
    -gt 大于
    -ge 大于等于
    -ne 不等于

  • 按照文件权限进行判断
    -r 有读权限
    -w 有写权限
    -x 有执行权限

  • 按照文件类型进行判断
    -f 文件存在并且是一个常规文件
    -e 文件存在
    -d 文件存在并是一个目录

if

if [ 条件判断式 ]
then
程序
elif
程序
fi

case

case $变量名 in
“value 1”)
程序1
;;
“value 2”)
程序2
;;
*)
;;
esac

for

for 变量 in value1 value2 value3
do
程序
done

或者
for((初始值，循环控制条件；变量变化))
do
程序
done

while

while [ 条件判断式 ]
do
程序
done

read读取控制台输入

read(选项)[参数]
选项:
-p:指定读取值时的提示符
-t:指定读取值时的等待时间(秒)，如果指定时间内没有输入，停止等待
变量：指定读取值的变量名

函数

  • 系统函数
    basename:返回完整路径最后/的部分，常用语获取文件名
    basename[pathname][suffix]
    basename[string][suffix]

dirname:返回完整路径最后/的前面部分，常用语返回路径部分

  • 自定义函数
    function funcname()
    {
    Action;
    [return int;]
    }
    调用直接写函数名：funcname[值]
