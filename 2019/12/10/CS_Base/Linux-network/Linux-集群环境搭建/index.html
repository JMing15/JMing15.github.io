Linux-集群环境搭建

发表于 2019-12-10 更新于 2019-12-12

基本环境步骤

  • 开启启动图形化界面关闭：
    vim /etc/inittab 将默认5级别改为3级别
  • 关闭防火墙，关闭SELinux
    1. 关闭防火墙：service iptables stop
      chkconfig iptables off
    2. 关闭SELinux: vim /etc/selinux/config 
      SELINUX=enforcing改为SELINUX=disabled（永久关闭SELINUX）
setenforce 0（临时关闭SELINUX）
# setenforce 1（临时打开SELINUX）
  • 配置IP地址
    1. vim /etc/sysconfig/network-scripts/ifcfg-eth0
      ONBOOT=no => ONBOOT=yes
      BOOTPROTO=dchp => BOOTPROTO=static
      添加IP和子网掩码：IPADDR=10.10.10.11 NETMASK=255.255.255.0
    2. 重启网路服务 service network restart
    3. #ifconfig 查看网络
  • 配置局域网YUM源
    1. 虚拟机设置CD/DVD(IDE)的设备状态设置为已连接
